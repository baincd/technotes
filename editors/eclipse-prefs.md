# Key Bindings
* Shift+Ctrl+P - Bind to Quick Access in Windows
  * Remap "Matching bracket/character/tag" commands to Shift+Ctrl+]
* Shift-Ctrl-F - Bind to Find Text in File in Windows
  * Remap "Format" commands to Ctrl+Alt+F
